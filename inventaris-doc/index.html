Gudang Gadang Documentation

Installing App.

  1. After unzip the download pack, you'll found a App Folder with all the files. And Database Folder
  2. Import Database File to your local/host database
  3. Move the App folder to your local/host database
  4. Once the files are done uploading go to www.yourdomainname.com/index.php
  5. Login with Administrator default
    Username : admin
    Password : admin

Role Login

Aplikasi ini memiliki 3 Hak akses untuk login. Yaitu : Administrator, Manajemen, Peminjam.
Berikut adalah spesifikasi mengenai hak akses tersebut.

Administrator

  • Dapat mengakses CRUD Barang
  • Dapat mengakses CRUD Supplier
  • Dapat mengakses CRUD Barang Masuk
  • Dapat mengakses CRUD Barang Keluar
  • Dapat mengakses CRUD Barang Pinjam
  • Dapat mengakses Manage Users
  • Dapat mengakses Log Login
  • Dapat mengakses Account Profiles

Manajemen

  • Dapat mengakses CRUD Barang
  • Dapat mengakses CRUD Supplier
  • Dapat mengakses CRUD Barang Masuk
  • Dapat mengakses CRUD Barang Keluar
  • Dapat melakukan Print Data
  • Dapat mengakses Account Profiles

Peminjam

  • Dapat mengakses CRUD Barang Pinjam
  • Dapat mengakses Account Profiles
